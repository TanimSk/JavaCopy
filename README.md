## CSERU Assignment

To run this project:
```
git clone https://github.com/TanimSk/JavaCopy
cd JavaCopy
```

For assignment 01:
```
cd assignment01
javac Copy.java
java Copy <sourceFile> <destinationFile>
```

For assignment 02:
```
cd assignment02
javac Copy2.java
java Copy2 <sourceFile/Dir> <destinationFile/Dir>
```
